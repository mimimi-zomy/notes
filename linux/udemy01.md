### 2.Linuxってなに？
#### Linuxとは？
- LinuxはOSの一種で、コンピュータを動かすための基本ソフト
- OSの種類
  - Windows
  - Mac OS X
  - Linux
  - Android　など
#### Linuxはどのように誕生したの？
- 1970年頃
　　 - UnixというOSが誕生
  - シンプルな小さいOSで大学や研究所を中心に普及
  - 有料。ライセンスの縛りも難しい
- 1985年〜
  - オープンソース版のUnixを開発するプロジェクトがスタート
  - ソースコードも公開されていたので、様々な派生版のUNIXが誕生
- 1991年
  - 当時大学生のリーナス・トーバスがLinuxを開発し、オープンソース化
  - UNIXを真似て、一からOSを作る
  - 最終的にも最も普及
  - **有料のUnixがLinuxという形で無料で使えるようになった**
